# midphysac

The goal of **midphysac** is to accompany the articles about putative effect of
high-level midlife physical activity maintained till elderly life on (super)aging
parameters such as *cognitive performance*, *functional independence*, and
*mental health*.

## Installation

You can install the current version of midphysac from GitHub via:

``` r
# install.packages("pak")
pak::pak("josefmana/midphysac")
```

## Georgi et al. ([2026](https://doi.org/10.1016/j.actpsy.2026.106478))
### Release v1.0.0

Compares effect of high physical activity levels sustained to later life to
average levels operationalised via population aimed at when sampling (COSACTIW
active women vs. NANOK general population of women) on measures of cognitive
superaging, functional independence and affective health.

Georgi, H., Mana, J., Schneiderova, M., Dadova, K., Ticha, Z., Vojtechova, I.,
... & Kopecek, M. (2026). Midlife physical activity as a link to better mental
health, functional status, and episodic memory in old-old women: Preliminary
study. *Acta Psychologica, 264*, 106478.
<https://doi.org/10.1016/j.actpsy.2026.106478>
